# Installation

> <b>Clone the repository:</b> <br/>
>'git clone https://github.com/Jeanpaulleon/ChartViewr.git'<br/><br/>
> <b>Go into the directory:</b> <br/>
>'cd ChartViewr'<br/><br/>
> <b>Update the node_modules packages:</b> <br/>
>'npm install'<br/> <br/>
> <b>Update the node_modules packages:</b> <br/>
>'npm run build'<br/> <br/>

# Launch
> <b>Run the application:</b><br/>
>'npm run start'<br/>
